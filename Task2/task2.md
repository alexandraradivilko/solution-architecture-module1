# Fit/Gap-анализ

## Список систем

| Система          | Статус системы | Функциональность                               | Статус функциональности | Комментарий                                                                                             |
| ---------------- | -------------- | ---------------------------------------------- | ----------------------- | ------------------------------------------------------------------------------------------------------- |
| ИС «Алмазы»      | Существующая   | Учёт алмазов                                   | Существующая            | Используется без изменений                                                                              |
| ИС «Алмазы»      | Существующая   | Отбор алмазов для цифрового аукциона           | Новая                   | Требуется добавить признаки отбора, статусы подготовки и формирование ведомостей для цифрового процесса |
| ИС «Алмазы»      | Существующая   | Планирование подготовки цифрового аукциона     | Новая                   | Требуется хранение информации о партиях, статусах оцифровки и готовности к публикации                   |
| ИС «Алмазы»      | Существующая   | Формирование данных для передачи в лабораторию | Новая                   | На первом этапе возможно формирование Excel-шаблонов вручную                                            |
| ИС «Аукцион»     | Существующая   | Проведение офлайн-аукционов                    | Существующая            | Используется как основа для цифрового процесса                                                          |
| ИС «Аукцион»     | Существующая   | Проведение онлайн-аукциона                     | Новая                   | Требуется поддержка удалённого участия клиентов                                                         |
| ИС «Аукцион»     | Существующая   | Публикация данных клиентам                     | Новая                   | На первом этапе через облачный сервис, далее — через внутреннюю инфраструктуру                          |
| ИС «Логистика»   | Существующая   | Перевозка алмазов покупателям                  | Существующая            | Используется без изменений                                                                              |
| ИС «Логистика»   | Существующая   | Отправка алмазов в лабораторию и возврат       | Новая                   | Требуется добавить новые маршруты, этапы и статусы перевозки                                            |
| ИС «Бухгалтерия» | Существующая   | Учёт оплаты перевозчиков                       | Существующая            | Используется без изменений                                                                              |
| ИС «Бухгалтерия» | Существующая   | Учёт услуг лаборатории                         | Новая                   | Требуется поддержка актирования и оплаты услуг лаборатории                                              |
| ИС «Бухгалтерия» | Существующая   | Учёт контрактов с облачным провайдером         | Новая                   | Требуется сопровождение новых договоров и расходов                                                      |
| ИС «Лаборатория» | Новая          | Оцифровка алмазов                              | Новая                   | Внешняя система подрядчика                                                                              |
| Облачный сервис  | Новый          | Хранение цифровых материалов                   | Новая                   | Требуется обеспечить защищённый доступ клиентов                                                         |

| ИС «Лаборатория» → ИС «Бухгалтерия» | Новая                  | Вручную на первом этапе, автоматически в целевом решении | Новые данные        | Передача актов и данных для оплаты услуг лаборатории              |
| ИС «Логистика» → ИС «Бухгалтерия»   | Существующая           | Автоматически                                            | Существующие данные | Передача данных для оплаты перевозчиков                           |

| ИС «Аукцион» → ИС «Бухгалтерия»     | Возможная доработка    | Автоматически                                            | Новые данные        | Передача итогов аукциона и финансовых данных                      |
| ИС «Аукцион» → ИС «Логистика»       | Возможная доработка    | Вручную на первом этапе, автоматически в целевом решении | Новые данные        | Передача информации о победителях торгов и необходимости доставки |
